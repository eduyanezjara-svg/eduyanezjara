Antes de comenzar

Comprueba que dispones de todo lo necesario para trabajar con ETS5 y programar tu instalación KNX.

Requisitos técnicos

  • Licencia activa de ETS5 con el número de dongle correspondiente.
  • Ordenador con Windows 10 o superior y permisos de administrador.
  • Interfaz de programación KNX (USB, IP o router KNX) conectada a la instalación.

Documentación previa

  • Plano eléctrico y topología propuesta (áreas, líneas, dispositivos).
  • Listado de funciones deseadas (iluminación, persianas, climatización, etc.).
  • Datos de catálogo o archivos .knxprod de los equipos a utilizar.

Buenas prácticas

  • Crear carpetas separadas para copias de seguridad y versiones del proyecto.
  • Definir una nomenclatura consistente para direcciones individuales y de grupo.
  • Documentar comentarios en cada objeto de grupo y dispositivo.
